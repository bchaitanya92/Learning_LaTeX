## üöÄ Getting Started

### 1. Prerequisites
- LaTeX distribution (TeX Live, MiKTeX, Overleaf, etc.)

### 2. Compile & View
Open your terminal or LaTeX editor and run:
```sh
pdflatex main.tex
```
Or open any `.tex` file in your LaTeX editor and build to generate the PDF.
